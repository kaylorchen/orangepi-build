## 本仓库是镜像生成仓库
镜像生成后，可以访问 http://59.46.137.178:8101/images/ 下载
默认的账号和密码分别是
```bash
hunter:jialin
root:jialin
```
Linux的升级方法是在系统中运行“reboot loader”或者按住maskrom按键启动进入Maskrom模式。
然后请查看当前工程中的upgrade_tool目录进行升级
